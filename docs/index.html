📖 標籤說明 （點擊展開）
三個標籤來自不同分析維度，可同時出現於同一張卡：

🟣 個股相位（BULL / BULL拉回 / RANGE / BEAR）
來源：個股日RS + 動能M + 加速度A 的百分位排名
意義：這支股票相對大盤是否持續強勢，BULL = RS加速中

↑ 結構標籤（主升段 / 突破準備 / 盤整 / 回檔 / 弱勢）
來源：MA5 > MA10 > MA20 均線排列 + AVWAP 支撐位置
意義：價格結構形態，主升段✓ = 均線多頭且站上所有AVWAP

🏷 類股相位（主升段 / 準備噴 / 主升回檔 / 整理觀察 / 空頭）
來源：類股輪動分析（RS/M/A 跨類股百分位 + 寬度）
意義：所屬類股目前在哪個輪動階段

同時出現的常見組合：
BULL + 主升段✓ + 主升回檔 = 個股強、結構健康，但類股稍微回檔，是觀察持續力的關鍵點
BULL + 突破準備 + 準備噴 = 類股與個股同步蓄勢，突破機率最高
BULL拉回 + 回檔 + 主升段 = 類股仍強，個股短期RS減速回檔，是最佳回測買點型態
RANGE + 盤整 + 整理觀察 = 個股與類股均在觀察等待，暫無明確方向
