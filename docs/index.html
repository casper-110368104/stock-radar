дёүеҖӢжЁҷзұӨдҫҶиҮӘдёҚеҗҢеҲҶжһҗз¶ӯеәҰпјҢеҸҜеҗҢжҷӮеҮәзҸҫж–јеҗҢдёҖејөеҚЎпјҡ

рҹҹЈ еҖӢиӮЎзӣёдҪҚпјҲBULL / BULLжӢүеӣһ / RANGE / BEARпјү
дҫҶжәҗпјҡеҖӢиӮЎж—ҘRS + еӢ•иғҪM + еҠ йҖҹеәҰA зҡ„зҷҫеҲҶдҪҚжҺ’еҗҚ
ж„Ҹзҫ©пјҡйҖҷж”ҜиӮЎзҘЁзӣёе°ҚеӨ§зӣӨжҳҜеҗҰжҢҒзәҢеј·еӢўпјҢBULL = RSеҠ йҖҹдёӯ

вҶ‘ зөҗж§ӢжЁҷзұӨпјҲдё»еҚҮж®ө / зӘҒз ҙжә–еӮҷ / зӣӨж•ҙ / еӣһжӘ” / ејұеӢўпјү
дҫҶжәҗпјҡMA5 > MA10 > MA20 еқҮз·ҡжҺ’еҲ— + AVWAP ж”Ҝж’җдҪҚзҪ®
ж„Ҹзҫ©пјҡеғ№ж јзөҗж§ӢеҪўж…ӢпјҢдё»еҚҮж®өвң“ = еқҮз·ҡеӨҡй ӯдё”з«ҷдёҠжүҖжңүAVWAP

рҹҸ· йЎһиӮЎзӣёдҪҚпјҲдё»еҚҮж®ө / жә–еӮҷеҷҙ / дё»еҚҮеӣһжӘ” / ж•ҙзҗҶи§ҖеҜҹ / з©әй ӯпјү
дҫҶжәҗпјҡйЎһиӮЎијӘеӢ•еҲҶжһҗпјҲRS/M/A и·ЁйЎһиӮЎзҷҫеҲҶдҪҚ + еҜ¬еәҰпјү
ж„Ҹзҫ©пјҡжүҖеұ¬йЎһиӮЎзӣ®еүҚеңЁе“ӘеҖӢијӘеӢ•йҡҺж®ө

еҗҢжҷӮеҮәзҸҫзҡ„еёёиҰӢзө„еҗҲпјҡ
BULL + дё»еҚҮж®өвң“ + дё»еҚҮеӣһжӘ” = еҖӢиӮЎеј·гҖҒзөҗж§ӢеҒҘеә·пјҢдҪҶйЎһиӮЎзЁҚеҫ®еӣһжӘ”пјҢжҳҜи§ҖеҜҹжҢҒзәҢеҠӣзҡ„й—ңйҚөй»һ
BULL + зӘҒз ҙжә–еӮҷ + жә–еӮҷеҷҙ = йЎһиӮЎиҲҮеҖӢиӮЎеҗҢжӯҘи“„еӢўпјҢзӘҒз ҙж©ҹзҺҮжңҖй«ҳ
BULLжӢүеӣһ + еӣһжӘ” + дё»еҚҮж®ө = йЎһиӮЎд»Қеј·пјҢеҖӢиӮЎзҹӯжңҹRSжёӣйҖҹеӣһжӘ”пјҢжҳҜжңҖдҪіеӣһжё¬иІ·й»һеһӢж…Ӣ
RANGE + зӣӨж•ҙ + ж•ҙзҗҶи§ҖеҜҹ = еҖӢиӮЎиҲҮйЎһиӮЎеқҮеңЁи§ҖеҜҹзӯүеҫ…пјҢжҡ«з„ЎжҳҺзўәж–№еҗ‘
