# Stock Radar 台股選股雷達

每日自動掃描台股，整合籌碼、基本面、量能、營收、相對強弱五大面向，給出綜合評分與買點訊號。

---

## 資料來源

| 資料 | 來源 | 說明 |
|------|------|------|
| 每日量價、類股指數 | TWSE 證交所公開 API | 免費，無須 token |
| 股價歷史、基本面 | Yahoo Finance (yfinance) | 免費，無須 token |
| 三大法人籌碼 | FinMind API | 免費個股查詢，需 token |
| 借券資料 | TWSE TWT93U | 免費，無須 token |
| 月營收 YoY | MOPS 公開資訊觀測站 / OpenData | 免費，無須 token |
| 新聞情緒 / 題材 | Google Gemini API | 需 API key，僅處理前 20 名個股 |

## 兩支掃描程式

| 程式 | 說明 |
|------|------|
| `fetch_stocks.py` | 主掃描器：全面評分 + 買點訊號 + Gemini 新聞情緒，每日執行 |
| `fetch_expansion.py` | 擴張掃描器：隨機抽取 500 支，純技術面訊號，不使用付費 API |

## 評分系統

每支股票的各面向分數皆為 **0～100 分**，最終再依下列權重加總為**綜合分數**：

```
綜合分數 = 籌碼面×35% + 基本面×30% + 量能面×25% + 月營收×5% + 相對強弱×5%
```

### 籌碼面（Chips）— 佔 35%

**基礎分 50 分**，視三大法人動向加減分。

#### 外資

| 條件 | 分數變動 |
|------|---------|
| 外資買超佔成交量 > 10% | +30 |
| 外資買超佔成交量 > 5% | +20 |
| 外資買超佔成交量 > 2% | +12 |
| 外資買超佔成交量 > 0% | +5 |
| 外資賣超佔成交量 < -5% | -15 |
| 外資賣超佔成交量 < -2% | -8 |

| 外資連買/賣天數 | 分數變動 |
|--------------|---------|
| 連買 ≥ 10 日 | +15 |
| 連買 ≥ 5 日 | +10 |
| 連買 ≥ 3 日 | +6 |
| 連買 ≥ 1 日 | +2 |
| 連賣 ≤ -5 日 | -12 |
| 連賣 ≤ -3 日 | -7 |
| 連賣 ≤ -1 日 | -3 |

#### 投信

| 條件 | 分數變動 |
|------|---------|
| 投信買超佔成交量 > 3% | +10 |
| 投信買超佔成交量 > 1% | +6 |
| 投信買超佔成交量 > 0% | +2 |
| 投信賣超佔成交量 < -1% | -5 |
| 連買 ≥ 5 日 | +5 |
| 連買 ≥ 3 日 | +3 |
| 連買 ≥ 1 日 | +1 |
| 連賣 ≤ -3 日 | -4 |

#### 自營商

| 條件 | 分數變動 |
|------|---------|
| 自營商買超佔成交量 > 2% | +3 |
| 自營商買超佔成交量 > 0% | +1 |
| 自營商賣超佔成交量 < -2% | -2 |

#### 借券（融券空頭壓力）

> 借券餘額佔流通股比例越高，代表空頭壓力越大

| 借券餘額佔流通股 | 分數變動 |
|--------------|---------|
| > 8% | -12 |
| > 5% | -8 |
| > 3% | -4 |
| = 0（無借券） | +2 |

| 借券趨勢（近3日） | 分數變動 |
|----------------|---------|
| 借券餘額增加 > 20% | -4（放空增加） |
| 借券餘額減少 > 20% | +3（空頭回補） |

| 借券賣出佔成交量 | 分數變動 |
|--------------|---------|
| > 3% | -4 |
| > 1% | -2 |

> 籌碼面最終分數**限制在 0～100 之間**。

### 基本面（Fundamental）— 佔 30%

**基礎分 50 分**，視 EPS 成長率、ROE、負債比加減分。

#### EPS 年增率

| 條件 | 分數變動 |
|------|---------|
| > 50% | +25 |
| > 30% | +18 |
| > 20% | +13 |
| > 10% | +8 |
| > 5% | +4 |
| > 0% | +1 |
| > -10% | -5 |
| > -30% | -12 |
| ≤ -30% | -20 |

#### ROE（股東權益報酬率）

| 條件 | 分數變動 |
|------|---------|
| > 25% | +20 |
| > 20% | +15 |
| > 15% | +10 |
| > 10% | +5 |
| > 5% | +1 |
| ≤ 0% | -10 |

#### 負債比

| 條件 | 分數變動 |
|------|---------|
| < 20% | +10 |
| < 30% | +6 |
| < 40% | +2 |
| > 70% | -8 |
| > 80% | -5（額外加重扣分） |

> 基本面最終分數**限制在 0～100 之間**。

### 量能面（Volume）— 佔 25%

比較近20日均量 vs 近60日均量（vr = vol20 / vol60）：

| 量比（vr） | 分數 |
|-----------|------|
| > 2.0 | 100 |
| > 1.5 | 85 |
| > 1.3 | 70 |
| > 1.1 | 55 |
| > 1.0 | 40 |
| > 0.8 | 25 |
| ≤ 0.8 | 10 |

> 若無法取得60日均量，改以當日量 vs 月均量（vol_month_ratio）代替，換算邏輯相同。

---

### 月營收 YoY（Revenue）— 佔 5%

比較本月營收與去年同期（年增率 %）：

| 年增率 | 分數 |
|--------|------|
| > 40% | 100 |
| > 20% | 85 |
| > 10% | 70 |
| > 5% | 60 |
| > 0% | 52 |
| > -5% | 45 |
| > -20% | 30 |
| ≤ -20% | 15 |

> 若無資料，預設給 50 分（中性），不影響整體評分。

---

### 相對強弱 RS（Relative Strength）— 佔 5%

與大盤比較的百分位排名（0～100）：

| RS 百分位 | 分數 |
|-----------|------|
| ≥ 90 | 100 |
| ≥ 80 | 85 |
| ≥ 70 | 70 |
| ≥ 50 | 55 |
| ≥ 30 | 40 |
| < 30 | max(10, 百分位 × 0.8) |

> 若無百分位資料但有 RS 絕對值，改以 RS 值換算（±0.20 對應 85/10 分）。

## 技術指標說明

### 均線（Moving Average）

| 均線 | 意義 |
|------|------|
| MA5 | 5日短期趨勢 |
| MA10 | 10日短期趨勢 |
| MA20 | 20日中期趨勢（月線） |
| MA60 | 60日中長期趨勢（季線） |

**多頭排列**：MA5 > MA10 > MA20，代表短中期趨勢皆向上。

### 產業輪動分析

用類股指數日報酬與大盤日報酬的差值（RS）計算，每日更新。
分為六個象限，代表各產業目前所處的輪動階段：

| 象限 | 條件 | 意義 |
|------|------|------|
| 準備噴 | 空頭結構，RS 剛翻正，動能轉強 | 底部轉折，注意追蹤 |
| 主升段 | 多頭結構，RS 強，動能加速 | 最強階段，主力資金湧入 |
| 主升回檔 | 多頭結構，短暫回檔，RS 趨勢仍上 | 回檔整理，可找買點 |
| 高檔震盪 | 多頭結構，動能放緩，趨勢轉弱 | 漲多整理，注意出場 |
| 空頭 | 空頭結構，RS 弱，動能持續下滑 | 全面弱勢，迴避 |
| 整理觀察 | 不符任一明確條件 | 過渡中性 |

## 買點訊號（fetch_stocks.py）

每支股票最多可同時觸發多個訊號，每個訊號包含：進場價、停損、目標價（固定 2:1 風報比）。

| 訊號名稱 | 觸發條件 | 強度 |
|----------|---------|------|
| 突破 | 收盤 > 20日高，量比 ≥ 1.5x，RS百分位 ≥ 70 | 強 |
| 假跌破 | 昨收跌破20日低，今收強力收復 | 強 |
| 均線回測 | 多頭排列（MA5>MA10>MA20），收盤距MA20在 3% 以內 | 中 |
| 強整再突 | 收盤緊貼20日高（距離 ≤ 5%），收盤 > MA5，RS百分位 ≥ 70 | 中 |
| 回測 | 縮量（量比 < 1.0）回測MA10，收盤 > MA20 | 中 |
| MA60支撐 | 收盤緊貼MA60（距離 ≤ 2%），RS百分位 ≥ 50 | 弱 |

> **停損設定原則**：以關鍵支撐（MA20、MA60、20日低等）為準，跌破即停損。
> **目標價計算**：進場價 + (進場價 - 停損) × 2，固定 2:1 風報比。

## 擴張掃描訊號（fetch_expansion.py）

對隨機抽取的 500 支股票進行純技術面訊號掃描（不使用籌碼資料）：

| 訊號名稱 | 觸發條件 | 強度 |
|----------|---------|------|
| 突破 | 收盤 > 20日高，量比 ≥ 1.5x | 強 |
| 假跌破 | 昨收跌破20日低，今收強力收復 | 強 |
| 均線回測 | 多頭排列（MA5>MA10>MA20），收盤距MA20在 3% 以內 | 中 |
| 強整再突 | 收盤緊貼20日高（距離 ≤ 5%），收盤 > MA5 | 中 |
| 縮量回測 | 縮量（量比 < 1.0）回測MA10，收盤 > MA20 | 中 |
| MA60支撐 | 收盤緊貼MA60（距離 ≤ 2%） | 弱 |

> 擴張掃描不計算 RS 百分位，故「突破」和「強整再突」無 RS 條件限制。

## 警示旗標

除買點訊號外，同時顯示以下警示供參考：

| 警示 | 觸發條件 |
|------|---------|
| 外資連賣 | 外資連續賣超 ≥ 5 日 |
| 高位量縮 | 股價在52週高點附近（位置 > 90%）但量能萎縮（量比 < 0.8） |
| EPS 衰退 | EPS 年增率 < -10% |

## 三法估值（合理價參考）

對每支股票提供保守、合理、樂觀三個估值，取三法中位數：

| 估值方法 | 說明 | 適用條件 |
|---------|------|---------|
| 歷史PE法 | EPS × 該股歷史本益比（低/中/高） | 有歷史PE資料 |
| PEG法 | EPS × EPS成長率（限 3%～30% 區間） | EPS成長率 > 3% |
| 市場PE法 | EPS × 台股平均PE（13x / 17x / 22x） | 所有股票 |

## 自動化排程

`.github/workflows/daily_fetch.yml` 設定每個工作日自動執行：

- `fetch_stocks.py`：主掃描，抓籌碼 + 基本面 + Gemini 新聞
- `fetch_expansion.py`：擴張掃描，隨機 500 股技術訊號

結果自動推送至 `docs/` 目錄，由 GitHub Pages 對外提供網頁服務。

## 快速開始

```bash
pip install -r requirements.txt

# 設定環境變數
export FINMIND_TOKEN=your_token
export GEMINI_API_KEY=your_key   # 可選

# 執行主掃描
python fetch_stocks.py

# 執行擴張掃描
python fetch_expansion.py
```
